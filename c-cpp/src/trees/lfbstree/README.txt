Implementation of a Lock-Free Unbalanced Binary Search Tree based on the paper
"Fast Concurrent Lock-Free Binary Search Trees" by Aravind Natarajan and 
Neeraj Mittal (UT Dallas)

This version of the code has been ported to work with synchrobench, and is current as of Feb 2015.

Features: 
  * Support for gcc versions > 4.6
  * Bug fixes

The code corresponding to the paper is available at:
  https://github.com/anataraja/lfbst.git
